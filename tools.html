Open-source remote sensing tools developed by the Silva lab and collaborators


1) rGEDI: An R Package for NASA's Global Ecosystem Dynamics Investigation (GEDI) Data Visualizing and Processing.


2) rLiDAR: An R Package for LiDAR Data Processing and Visualization.


3) Treetop: A Shiny-based Application for Extracting Forest Information from LiDAR data.


4) ForestGapR: An R Package for Airborne Laser Scanning-derived Tropical Forest Gaps Analysis.
